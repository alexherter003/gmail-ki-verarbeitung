# Gmail KI-Verarbeitung

## Projektbeschreibung
Automatische Verarbeitung eingehender E-Mails mit ChatGPT-Analyse 
und automatischer Sortierung in Notion-Datenbanken.

## Ablauf
1. Gmail überwacht Posteingang auf neue E-Mails
2. ChatGPT analysiert und klassifiziert jede E-Mail
3. JSON-Modul verarbeitet die KI-Antwort
4. Router entscheidet basierend auf KI-Bewertung
5. Automatisch verarbeitbare E-Mails → Notion Datenbank 1
6. E-Mails die manuelle Prüfung brauchen → Notion Datenbank 2

## Tools
- Make.com
- Gmail
- OpenAI (ChatGPT)
- Notion

## Ziel
Manuelle E-Mail-Sortierung automatisieren und 
Reaktionszeit auf wichtige Anfragen reduzieren.
